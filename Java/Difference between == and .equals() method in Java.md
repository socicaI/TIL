# equals()와 == 연산자 공통점과 차이점
## equals()
* 메서드이다.
* 객체의 값을 비교할 때 사용
* 문자열의 데이터/내용을 기반으로 주어진 두 문자열을 비교

## == 연산자
* 연산자(operator)이다.
    * 연산자(operator): 주어진 식을 계산하여 결과를 얻어내는 과정을 연산이라 하고, 연산을 수행하는 기호를 뜻함
* 비교 대상인 두 객체가 동일한 메모리 위치를 가리키는지 확인
